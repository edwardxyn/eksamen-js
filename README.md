# Erklæring

Jeg har brukt KI på samme måte som jeg fikk beskjed å bruke google/youtube når jeg startet på Gokstadakademiet. Jeg har brukt KI til å lære, forklare, og inspirere. I steden for å browse nettet og sider som "Stack Overflow", har jeg sendt prompts til claude.ai. Jeg har brukt claude til å hjelpe med "helper functions"/holde koden ryddig,  finne default image, styling(farger og utrykk), emoticons, samt å forklare vanskeligere konsepter når det kommer til JavaScript.
Jeg går god for å ha skrevet og forstått alt i repositoriet.
